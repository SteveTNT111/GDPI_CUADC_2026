# 给机载电脑 Codex 的视觉伺服任务包

> 阅读对象：机载 NUC 上的 Codex。
> 今天目标：不要重写全系统，只补齐“识别到目标后，飞机低速移动直到对准，然后交给现有抛投逻辑”的最小闭环。

---

## 0. 先做地面测试，不要先飞实机

学长最新指导的核心是：今天第一步不是直接让飞机飞去瞄准，而是先把“检测输出”和“A/B 抛投器瞄准偏置”测清楚。

P0 必须先完成：

```text
1. 不飞实机，先地面跑通 detector_node.py 与半自动投放测试脚本通信
2. 确认 detector_node.py 能输出桶数量、目标位置、飞机高度等必要信息
3. 测出 A、B 两个抛投器相对于相机系/画面中心的偏差值
4. 半自动脚本收到目标后，在终端询问是否切入自动瞄准
5. 人拿遥控器防意外，确认安全后按 Enter，脚本才进入自动瞄准
```

重要：不能默认用相机画面中心点瞄准抛投。

```text
如果用 A 抛投器投放：
  实际瞄准点 = 相机中心点 + A 抛投器偏置

如果用 B 抛投器投放：
  实际瞄准点 = 相机中心点 + B 抛投器偏置
```

也就是说，视觉伺服的目标不是简单让桶到画面正中心，而是让桶到“当前抛投器对应的瞄准点”。例如学长提到：用 A 抛投器时，可能需要用画面上面一点的位置来瞄准。

建议先把测得的偏置写成参数：

```text
~dropper_a_offset_x_m
~dropper_a_offset_y_m
~dropper_b_offset_x_m
~dropper_b_offset_y_m
```

或如果先用像素测试：

```text
~dropper_a_offset_px_x
~dropper_a_offset_px_y
~dropper_b_offset_px_x
~dropper_b_offset_px_y
```

半自动测试脚本的目标流程：

```text
启动脚本
-> 与 detector_node.py 通信
-> 接收识别到的桶信息
-> 终端提示：是否切入自动瞄准
-> 人拿遥控器准备接管
-> 按 Enter
-> 飞机/仿真/地面测试逻辑进入自动瞄准
-> 对准当前抛投器的瞄准点
```

学长口中的 `detect_node.py`，在本仓库里大概率对应实际文件：

```text
代码/NUC上面的测试功能包（不可以随便修改,可以复制里面的内容）/cuadc_src/scripts/detector_node.py
```

## 6. 建议实现方案

最稳妥路线：

```text
先写独立 semi_auto_drop_test.py 或 visual_servo_test.py 验证检测输出、A/B 偏置和方向
方向正确后，再把逻辑并入 main.py
```

如果时间紧，也可以直接在 `main.py` 中补最小函数，但要加开关参数，默认关闭。

建议参数：

```text
~enable_visual_servo      默认 false
~visual_servo_kp          默认 0.4
~visual_servo_max_speed   默认 0.20
~visual_servo_deadband_m  默认 0.10
~visual_servo_timeout_s   默认 0.50
~active_dropper           默认 A，可选 A / B
~dropper_a_aim_camera_x_m 先测量
~dropper_a_aim_camera_y_m 先测量
~dropper_b_aim_camera_x_m 先测量
~dropper_b_aim_camera_y_m 先测量
```

最小控制逻辑：

```python
def _limit(value, limit):
    return max(-limit, min(limit, value))

def _publish_zero_velocity(self):
    msg = TwistStamped()
    msg.header.stamp = rospy.Time.now()
    msg.twist.linear.x = 0.0
    msg.twist.linear.y = 0.0
    msg.twist.linear.z = 0.0
    self.vel_pub.publish(msg)

def _check_visual_servo(self):
    if not self.enable_visual_servo:
        return

    if self.current_state.mode != "GUIDED" or not self.current_state.armed:
        self._publish_zero_velocity()
        return

    with self.detect_lock:
        det = self.latest_detection

    if det is None or not det.position_valid:
        self._publish_zero_velocity()
        self._aligned_since = None
        return

    # 不要默认瞄准相机中心。
    # 先减去当前抛投器对应的相机系瞄准点偏置。
    # aim_camera_y_m 为负时，表示瞄准点在画面上方。
    aim_x = self.dropper_a_aim_camera_x_m
    aim_y = self.dropper_a_aim_camera_y_m
    if self.active_dropper == "B":
        aim_x = self.dropper_b_aim_camera_x_m
        aim_y = self.dropper_b_aim_camera_y_m

    err_cam_x = float(det.camera_x_m) - aim_x
    err_cam_y = float(det.camera_y_m) - aim_y

    # 相机系误差 -> 机体系速度误差参考：
    # body_x 前向 = -camera_y_m
    # body_y 右向 =  camera_x_m
    err_forward = -err_cam_y
    err_right = err_cam_x

    if abs(err_forward) < self.visual_servo_deadband_m and abs(err_right) < self.visual_servo_deadband_m:
        self._publish_zero_velocity()
        return

    vx = _limit(self.visual_servo_kp * err_forward, self.visual_servo_max_speed)
    vy = _limit(self.visual_servo_kp * err_right, self.visual_servo_max_speed)

    msg = TwistStamped()
    msg.header.stamp = rospy.Time.now()
    msg.twist.linear.x = vx
    msg.twist.linear.y = vy
    msg.twist.linear.z = 0.0
    self.vel_pub.publish(msg)
```

重要：`/mavros/setpoint_velocity/cmd_vel` 的速度方向到底是机体系还是 ENU/local 系，要在 NUC 上确认。若方向反了，只改符号，不要提高速度硬试。

## 8. 今天测试顺序

P0：

```text
1. 不飞实机，先地面确认 detector_node.py 与半自动测试脚本通信
2. 确认终端能打印：桶数量、camera_x/y/z、NED/WGS84、飞机高度
3. 测量 A 抛投器相对相机中心/相机系的瞄准偏置
4. 测量 B 抛投器相对相机中心/相机系的瞄准偏置
5. 半自动脚本必须先询问是否切入自动瞄准，按 Enter 后才动作
6. 桨叶拆下，地面确认目标丢失时速度立刻变 0
7. 单独发布很小速度，确认 MAVROS 速度话题有效
8. 只测 X 方向：目标在画面右边时，飞机是否向右修正
9. 只测 Y 方向：目标在画面上/下时，飞机是否向正确方向修正
10. 对准 A/B 当前瞄准点阈值内，确认速度为 0
```

P1：

```text
1. 接入 main.py 的 MISSION 状态
2. 开启 enable_visual_servo
3. 接入 active_dropper 和 A/B 瞄准偏置
4. 保持 enable_auto_drop 可控，不要和 auto_drop_node.py 重复触发
5. 低高度、低速度、空旷场地试飞
```

P2：

```text
1. 加滑动平均或限加速度
2. 处理多目标选择
3. 加投放器 A/B 偏置补偿
4. 合入完整比赛状态机
```

## 9. 安全底线

```text
最大速度先限制在 0.10 到 0.20 m/s

```
