## 🔧 技术方案与分工

### 仿真（伍尚京 + 刘辰宇）

> 先搭通仿真 → 视觉和控制组在仿真里开发 → 最后上实机

- **固件已定**：AP（ArduPilot），遥控器能用 + SITL 同样成熟
- AP SITL + Gazebo 链路搭建（`sim_vehicle.py -v ArduCopter`）
- MAVROS 连接假飞控
- Gazebo 假摄像头输出

### 视觉（潘科宇 + 赖斯远 + 陈智勇 + 伍尚京）

> 当前进度：**CV 黄色圆检测已部署上机 ✅**。YOLOv8 等圆筒到了再训。
>
> **这周目标：不管用什么方法，能检测到替代标识物并输出位置就行。**

- **陈智勇**：D435i 上机安装 + SDK 调通 + 传统CV黄色圆检测 ✅ **已上机**
- **赖斯远**：YOLOv8 检测地面桶 + D435i 深度融合（等圆筒到货后训）
- **潘科宇**：传统CV辅助 + 控制接口对接（黑色NUC坏了，先共用或拉网线）
- **伍尚京**：把控视觉方案、解决疑难
- 先出 2D 检测 + 深度 → (x,y,z)，精度不够再上位姿估计

### 控制（刘辰宇 + 潘科宇 + 郑宇）

- **固件已定：AP（ArduPilot）**，Guided 模式控飞机
- MAVROS 读取 RTK/GPS
- 机载电脑切换飞行模式（Offboard / Guided）
- 发坐标/速度/加速度指令
- **交付**：ROS 节点文档，视觉组拿着就能联调

### 运营（林悦超）

- **7/5 前**：完成比赛报名
- 采购耗材（3D打印材料、接线端子等）
- 跟进雷迅赞助审核进度
- 拍摄装机/试飞素材
