## 🔧 技术方案

### 视觉（cuadc_src）

> 当前状态：✅ 已完成，见 [[src_README|src_README.md]]

- **检测**：YOLOv8n（CPU 18.6FPS）/ YOLOv8s（8.6FPS），天津大学开源数据集训练
- **坐标变换**：相机系 → tf2 → 机体 → ENU → WGS84 经纬高
- **自动投放**：检测框中心与画面中心偏差 < 阈值 → 自动触发舵机
- **飞行录像**：解锁自动录，叠加 OSD 飞行数据，降落自动停
- 代码仓库：`cuadc_src/`（7 个节点 + 6 个 launch 文件）

### 飞控（ArduPilot + MAVROS）

- 固件：AP（ArduPilot）
- 模式：GUIDED 切换 + setpoint 位置控制
- 通信：MAVROS（`/mavros/setpoint_position/local`）
- 状态机：INIT → PREARM → ARMED → TAKEOFF → HOLD → MISSION → LAND/RTL

### 舵机投放

- 控制方式：MAVLink `MAV_CMD_DO_SET_SERVO`
- 触发逻辑：对准检测 → 像素偏差 < threshold → 舵机打开 → 保持 → 复位 → 冷却
- 参数：release_pwm=1900, reset_pwm=1100, cooldown=3s

## ⚠️ 关键风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| **🔴 Setpoint 控制不通** | 飞机飞不到桶上方 | 最优先解决，简化策略：先悬停定点，再手动微调 |
| **🔴 外场下雨** | 无法实飞测试 | 室内地面测试 + 仿真先行，雨停立刻冲外场 |
| **底板挡相机** | 低高度投水时水瓶遮住镜头 | 调整相机安装位置，或提高投放最低高度 |
| **机组二不熟代码** | 现场跑不起来 | 7/12 前集中培训，写死 launch 文件参数 |
| **双机同场干扰** | IR 投影器互相干扰深度 | 户外阳光淹投影，影响不大；必要时遮一台 |
| **炸机** | 没飞机用 | 一号机稳扎稳打，二号机尽快搭好做备份 |
| **交通/住宿** | 到不了赛场 | 已讨论卧铺方案，林悦超跟进订票 |
